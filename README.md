## 中文

### 模组简介

**模组为我的世界添加了7种合成类型, 支持使用[KubeJS](https://kubejs.com/wiki/tutorials)和[CraftTweaker](https://docs.blamejared.com/1.20.1)编写。**

### 合成类型

1. **流体融合** - 将多个物品丢入指定流体中融合
2. **流体转换** - 在指定流体中转换单个物品
3. **虚空转换** - 在虚空深度（Y轴-65）进行转换
4. **火焰转换** - 使用火焰类型进行转换
5. **闪电转换** - 通过闪电进行转换
6. **爆炸转换** - 利用爆炸进行转换
7. **粉碎转换** - 使用重力方块砸落进行转换
